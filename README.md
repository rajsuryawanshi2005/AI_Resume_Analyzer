# AI_Resume_Analyzer
AI Resume Analyzer built with Python Flask that analyzes resumes, compares skills with job descriptions, calculates match score, and provides AI suggestions with visual charts.

AI Resume Analyzer is a Flask-based web application that analyzes resumes and compares them with job descriptions.

## Features

- Resume Match Score (0–100%)
- Skill Extraction
- Missing Skills Detection
- AI Resume Suggestions
- Skill Match Visualization (Chart.js)
- Beautiful Web UI

## Tech Stack

- Python
- Flask
- HTML/CSS
- Chart.js
- PDFPlumber
- DOCX Parser

## How to Run

1. Clone repository
2. Install dependencies

pip install -r requirements.txt

3. Run app
python app.py

4. Open browser
http://127.0.0.1:5000
